Sistema Gerencial
Desenvolvido por Bruno Romero de Alencar

Usado Linguagem de Programação Java
Banco de Dados MySQL
Ainda em Desenvolvimento.
Módulos 
-Cadastros
	+Usuário
	+Marca Produto
	+Linha Produto
	+Produto
	+Fornecedor
	+Cliente
-Venda
	Obs: Ao Fazer venda é feito a baixa do produto no estoque, o mesmo acontece 
	quando a venda é cancelada, a quantidade de cada item da venda é estornado
	para o estoque.
-Consulta Movimentação de Venda
	Obs: Consulta vendas feita no sistema, tem a opção de emitir relatórios gerenciais
	e cancelar venda.
-Caixa
	Obs: Ao finalizar uma venda é lançado o registro da venda no caixa, quando 
	cancelado a venda é lançado também um registro no caixa com o valor da venda,
	porém o valor da venda será lançado como negativo, ou seja uma saída de caixa.
